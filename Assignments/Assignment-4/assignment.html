Feature of HTML-5

1.Semantic Tags

HTML5 introduces semantic tags like <header>, <nav>, <article>, and <footer>, which provide better structure and meaning to web content. These tags improve accessibility and help search engines understand the content.

2. Video and Audio Support

HTML5 includes native support for embedding audio and video content using the <video> and <audio> elements. This eliminates the need for third-party plugins like Flash and makes multimedia content more accessible and responsive.

3. Form Enhancements

HTML5 introduces new input types and attributes for forms, such as <input type="email"> and <input type="date">, making it easier to collect specific data from users. Additionally, client-side validation and the <datalist> element improve the user experience.
