HTML defines the content and structure of your website

Comments in HTML serve the purpose of providing explanatory or descriptive text within the HTML code that is not meant to be displayed in the web browser. They are primarily used for documentation, notes to other developers, or for temporarily removing code without deleting it. Comments are ignored by web browsers and are invisible to the end-users. They are enclosed within delimiters
